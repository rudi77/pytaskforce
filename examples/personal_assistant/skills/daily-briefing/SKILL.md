# Daily Briefing Skill

## Objective
Summarize the user's day using email, calendar, and task data.

## Workflow
1. List today's calendar events.
2. List open tasks and highlight priority items.
3. List inbox emails (status=inbox) and summarize top items.
4. Provide a short actionable briefing with time-boxed recommendations.

## Output Format
- **Schedule**: Bullet list of events with times.
- **Top Tasks**: 3-5 critical tasks.
- **Inbox Highlights**: 3-5 emails with suggested action.
- **Recommendations**: 2-3 next-step suggestions.
