# Calendar Assist Skill

## Objective
Help schedule and prepare for meetings with clear summaries and follow-ups.

## Workflow
1. List upcoming events for the next 7 days.
2. Identify gaps for scheduling and propose time slots.
3. Provide prep notes with attendee focus and agenda suggestions.
4. Recommend follow-up tasks for each meeting.

## Output Format
- **Upcoming Meetings**: Date/time, title, attendees.
- **Open Slots**: 2-3 suggested time windows.
- **Prep Notes**: Agenda bullets and key questions.
