Core

ReAct Loop & Planning Strategies

ReAct: Reason + Act

1. THOUGHT
LLM analysiert Aufgabe und plant naechsten Schritt
▼
2. ACTION
Tool-Auswahl und Ausfuehrung (native tool calling)
▼
3. OBSERVATION
Tool-Ergebnis wird analysiert
▼ repeat bis Mission erledigt

4 Planning Strategies

NativeReAct Default

Purer ReAct-Loop: Thought -> Action -> Observation

PlanAndExecute

Plan erstellen, dann Schritte sequentiell ausfuehren

PlanAndReact Hybrid

Plan erstellen, ReAct innerhalb jedes Schritts

SPAR

Sense -> Plan -> Act -> Reflect (iterative Verfeinerung)

Dynamic LLM Routing

Jede Phase nutzt optimales Modell:
planning reasoning Strong Model · acting Standard · summarizing Fast/Cheap
